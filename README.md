# 📦 Supply Chain Delivery Analysis

## 📌 Objective
Analyze delivery performance and identify inefficiencies in shipping operations.

## 🛠 Tools Used
- Python (Pandas)
- Power BI

## 📊 Key Steps
- Cleaned and processed ~10,000 records
- Converted date columns and handled inconsistencies
- Created "Delivery Days" feature

## 🔍 Key Insights
- Central region showed slightly higher delivery delays
- Standard Class shipping had the highest delivery time (~5 days)
- Standard Class also had the lowest profitability

## 📈 Dashboard
Built an interactive Power BI dashboard to visualize delivery performance and profit trends.

## 🎯 Conclusion
Shipping mode impacts delivery performance more than region. Optimizing shipping strategy can improve both speed and profitability.
